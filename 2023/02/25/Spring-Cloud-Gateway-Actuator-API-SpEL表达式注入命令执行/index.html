漏洞简述

Spring Cloud Gateway是Spring中的一个API网关。其3.1.0及3.0.6版本（包含）以前存在一处SpEL表达式注入漏洞，当攻击者可以访问Actuator API的情况下，将可以利用该漏洞执行任意命令。

影响范围

Spring Cloud Gateway 3.1.x - 3.1.1

Spring Cloud Gateway 3.0.x - 3.0.7

旧的、不受支持的版本也会受到影响
