neo-regreg代理

netsh做端口映射

  1. netsh做端口映射：netsh interface portproxy add v4tov4 listenport=7777 connectaddress=192.168.138.130 connectport=80
  2. 查看映射: netsh interface portproxy show all

使用netsh进行msf的代理

（就是server2的木马访问不到外网，需要中间人代理）

  1. msf一个木马，把木马放进内网主机（生成的木马lhost是中间机的）
  2. 在中间机上做netsh端口映射
  3. 启动木马即可
